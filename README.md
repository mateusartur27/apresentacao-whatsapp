# Apresentacao WhatsApp: Arquitetura, Stack e Kafka

## Visao geral
Este projeto e uma apresentacao web sobre como o WhatsApp funciona em escala global.

O material esta organizado em tres eixos:
- Arquitetura de sistema (modelo C4)
- Stack de tecnologia
- Deep dive de Apache Kafka no contexto do WhatsApp

A apresentacao foi montada em HTML, CSS e JavaScript puro, com foco didatico e visual.

## Conteudo principal
- C4 Nivel 1 (contexto do sistema)
- C4 Nivel 2 (visao de conteineres)
- Visao consolidada da arquitetura
- Stack tecnologica (backend, dados, criptografia, realtime, cliente)
- Pipeline de eventos com Kafka (topicos, casos de uso e padroes)

## Estrutura do projeto
- index.html: apresentacao principal
- stack.html: detalhamento da stack de tecnologia
- kafka.html: deep dive sobre Apache Kafka
- assets/: diagramas e imagens de apoio

## Como executar localmente
1. Baixe ou clone este repositorio.
2. Abra a pasta no VS Code.
3. Abra o arquivo index.html no navegador.

Opcional:
- Use a extensao Live Server para navegar com recarregamento automatico.

## Navegacao recomendada
1. Comece por index.html para a visao geral.
2. Acesse stack.html para detalhamento tecnico da stack.
3. Finalize em kafka.html para aprofundar o fluxo de eventos.

## Objetivo academico
Este material foi construido para estudo e apresentacao tecnica, com base em fontes publicas sobre arquitetura distribuida, engenharia de sistemas e processamento de eventos.
